# Simple Calculator in C++

This is a basic C++ program that takes two numbers as input and prints their sum.

## How It Works
- The program prompts the user to enter two numbers.
- It passes the numbers to the `add` function.
- The function returns the sum of the two numbers.
- The result is displayed on the screen.

## Code Explanation
- `add(int a, int b)`: A function that returns the sum of two integers.
- `main()`: The main function takes user input, calls the `add` function, and prints the result.
